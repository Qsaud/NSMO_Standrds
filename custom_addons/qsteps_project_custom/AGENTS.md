# AGENTS.md (qsteps_project_custom)

This is the active customization addon.

- All new Odoo customizations should go here.
- Use inheritance.
- Keep the module compatible with Odoo 18 Community.
- Keep changes simple.
- Do not duplicate the original Project module.

Follow this structure:
- `models/` for Python model extensions
- `views/` for backend and portal XML views
- `security/` for access rights and rules
- `report/` for printable reports
- `controllers/` only when necessary
